# 💼 Portfólio - Mel Letícia Assis

Este projeto é um **portfólio pessoal desenvolvido em HTML, CSS e JavaScript**, criado para apresentar informações profissionais, habilidades, cursos e formas de contato.

O objetivo do projeto é demonstrar meus conhecimentos iniciais em **desenvolvimento web** e servir como uma apresentação profissional para recrutadores e outros desenvolvedores.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 📌 Funcionalidades

* Apresentação pessoal (Quem sou)
* Exibição de habilidades
* Informações sobre cursos realizados
* Área de contato
* Interface simples e intuitiva
* Navegação dinâmica com JavaScript

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de:

* Praticar **estruturação de páginas web**
* Aplicar **estilização com CSS**
* Utilizar **JavaScript para interatividade**
* Criar um **portfólio profissional inicial**
